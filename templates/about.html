About CryptoPulse

Project Overview

CryptoPulse is a real-time cryptocurrency tracking application that provides live price updates, portfolio management, and price alerts for popular cryptocurrencies.

Key Features
  • Real-time Price Tracking: Live updates for Bitcoin, Ethereum, Solana, and more
  • User Authentication: Secure login system with role-based access
  • Price Alerts: Set custom price thresholds with visual indicators
  • Live Search: Filter and search through tracked cryptocurrencies
  • Admin Panel: Manage tracked coins and view user statistics
  • Portfolio Tracking: Virtual trading with fake money (coming soon)
Technology Stack
Backend
  • Python Flask - Web framework
  • Flask Sessions - User authentication
  • Werkzeug - Password hashing
  • Requests - API calls
Frontend
  • HTML5 & CSS3
  • Bootstrap 5 - UI framework
  • JavaScript - Dynamic interactions
  • Chart.js - Data visualization
Data Source

Real-time cryptocurrency data is fetched from the CoinGecko API, which provides reliable and up-to-date market information for thousands of cryptocurrencies.

How It Works
  1. Data Fetching: The application makes HTTP requests to CoinGecko API every 30 seconds
  2. Real-time Updates: JavaScript automatically refreshes price data without page reload
  3. User Management: Flask sessions handle secure user authentication and authorization
  4. Price Alerts: Client-side JavaScript monitors prices and triggers visual alerts
